# 🏥 Clinic Management System

A full-stack Clinic Management System designed to manage clinics, doctors, patients, and appointments in a structured and scalable way.

The system allows patients to book appointments, and administrators to manage clinics, doctors, schedules, and patient data efficiently.

---

# 🎯 Project Idea

This project was built to digitize the traditional clinic workflow by:

- Eliminating manual appointment booking
- Preventing schedule conflicts between patients
- Centralizing patient and doctor data
- Supporting multiple clinics with different schedules

---

# 🏗 System Architecture

Frontend (Next.js)
        ↓
Backend API (Node.js / Express)
        ↓
Database (MongoDB)

---

# 🧠 Core System Flow

1. Authentication Flow
- User logs in or registers
- JWT generated by backend
- Refresh token stored in httpOnly cookie
- Middleware protects dashboard routes

2. Appointment Flow
- Select clinic → doctor → time
- System checks availability
- Saves appointment if slot is free

3. Admin Flow
- Manage doctors
- Manage patients
- Monitor appointments
